🛠️ System Hijacker v3.0 Changelog
🔒 WPA2/WPA3 Mandatory Skip Logic
Auto-Filter: Added a real-time length check to skip any passwords under 8 digits.
Efficiency: The script now uses a continue statement to jump to the next key without calling the test_wifi function, preventing wasted time on invalid handshake attempts.
🛰️ Real-Time Protocol Detection
Auto-Scan: Integrated a get_protocol_details function that scans the target SSID before starting the injection.
Dynamic AKM Mapping: The tool now identifies if the target is WPA, WPA2-PSK, or WPA3 (SAE) and applies the correct authentication key management (AKM) type automatically.
📊 UI & Visual Overhaul
Typewriter Effect: Applied the slow-typing "hacker" visual to all critical status lines, success messages, and system warnings for a dramatic effect.
Counter Fix: Implemented a "Clean Counter" using carriage returns (\r) and trailing space padding. This prevents numbers from overlapping when skipping short passwords.
🔊 Audio Feedback System
Success Alert: Integrated a Windows Beep (winsound) that triggers the moment a password is found. Added a universal terminal bell (\a) fallback for non-Windows systems.
📡 Realtek AP Platform Optimization
SDK Identification: Tuned the scanning logic to recognize generic Realtek AP-Router SDK profiles (Model: EV-2010-09-20).
Priority PINs: Added support for identifying and testing common Realtek-specific default PINs (e.g., 12345670, 77752499, 56730298).
☣️ Success Sequence Update
Compromise Warning: After a successful crack, the tool now displays a full-sequence typewriter warning: "password compromised !!!" and "Re-treat immediately".
Auto-Logging: All successful cracks are timestamped and saved to cracked_passwords.txt automatically.
🛑 Safe Exit
Stealth Close: Updated the KeyboardInterrupt message to "It's not me!" for a cleaner, themed exit when hitting Ctrl+C
Added auto scanning no need to enter essid manually
